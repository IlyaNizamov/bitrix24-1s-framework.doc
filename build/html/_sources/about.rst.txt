Framework для интеграции 1С и Битрикс24
=======================================

**Текущая версия 1.0.6**

**Возможности**

* Базовый модуль по работе с API Битрикс24
* Аутентификация OAuth 2.0
* Консоль разработчика
* Помошник начальной настроки
* Парсер контактной информации
* Интеграция с сущностью "Лиды"
* Интеграция с сущностью "Контакты"
* Интеграция с сущностью "Компании"
* Интеграция с сущностью "Сделки"
* Интеграция с сущностью "Товары"
* Интеграция с сущностью "Разделы товаров"
